Reflectie Jens Blok 2B:


Onderwerpen
Dit semester heb ik geleerd over arrays, wat data structures zijn die een sequentie van elementen van hetzelfde type opslaan. 
Ook ben ik aan de slag gegaan met pointers, dat zijn variabelen  die het geheugenadres van een andere variabele opslaan. Om een pointer in C++ te declareren, heb ik de *-operator gebruikt. Ook kan ik nu een pointer dereferencen.
Polymorphisme is ook aan bod gekomen, dat is een mechanisme waarbij een object meerdere vormen kan aannemen. In C++ kun je dit bereiken via inheritance en virtual functions.
Het meest heb ik mijn tanden gezet in 2D arrays, dit zijn arrays et 2 dimensies. Ik heb dit gebruikt om dynamisch oscillatoren aan te kunnen maken en hier vervolgens methods van aan te roepen.
